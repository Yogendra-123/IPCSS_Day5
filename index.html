4. What are CSS combinators ?

In CSS a combinator explains the relationship between the selectors.A CSS selector can contain more than one simple selector. Between the simple selectors, we can include a combinator.There are four different combinators in CSS:

  1. Descendant selector (space)
  2. Child selector (>)
  3. Adjacent sibling selector (+)
  4. General sibling selector (~)

5. What are pseudo selectors in CSS.

In CSS A pseudo-selector is used to define a special state of an element.Some of the mostly used selectors are . hover, active , visited , link , first-child , nth-child. etc
